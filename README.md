Проект позволяет вычислить значение функции f(x) = ln(1-x) на промежутке [-1 ; 1) с помощью ряда Тейлора. Точность вычисления определяется параметром k (подсчет суммы ряда прекращается, когда очередное слагаемое ряда становится по модулю меньше 10^(-k)).
Вводимые данные:
- Параметр k (положительное целое число, задающее порядок точности слагаемого)
- Значение x (в пределах [-1; 1))
Результат работы программы:
- Форматированный вывод числа k в десятичной, восьмеричной и шестнадцатеричной системах счисления
- Сумма ряда, которая вычисляется до тех пор, пока модуль очередного слагаемого не станет меньше 10^(-k)
- Сравнение результата со значением, вычисленным стандартной функцией Math.log()
Структура проекта:
"Main.java" — класс, отвечающий за ввод данных, форматированный вывод числа k, вызов метода вычисления значения функции с помощью ряда Тейлора
"Logarithm.java" — класс для вычисления значения функции с помощью ряда Тейлора
